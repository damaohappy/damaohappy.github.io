排序算法思想的应用

Posted on 2020-01-17 Edited on 2020-01-18

题目来源：Leetcode-75.Sort Colors

Facebook；Microsoft；PocketGems

题目难度：Medium

题目描述：Given an array with n objects colored red, white or blue, sort them in-place so that objects of the same color are adjacent, with the colors in the order red, white and blue.

Here, we will use the integers 0, 1, and 2 to represent the color red, white, and blue respectively.

Note: You are not suppose to use the library's sort function for this problem.

题目大意：给定数组nums[]，其中只有0,1,2三种数，排序nums[]。

样例：

Input: [2,0,2,1,1,0] Output: [0,0,1,1,2,2]

计数排序

由于给出数据的取值只有0、1和2三种可能，所以我们完全可以扫描一遍数组，统计这三种数字的个数，再以此填入原数组中。

三路快速排序

数组nums[]分为三个部分，nums[0...zeros]为0，nums[twos...n-1]为2， 中间部分为1。当遍历到某个元素nums[i]，该元素为0时则与nums[zeros + 1]交换，zeros与i自增；该nums[i]为1时，i自增；nums[i]为2时，与nums[twos - 1]交换，twos自减，i自增。

上述过程实际上就是三路快排的partition的过程。
