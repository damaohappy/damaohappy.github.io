C++实现带权重生成随机数

Posted on 2019-04-25 Edited on 2020-01-17

题目来源：Leetcode-528.Random Pick with Weight

题目难度：Medium

题目描述：

Given an array w of positive integers, where w[i] describes the weight of index i, write a function pickIndex which randomly picks an index in proportion to its weight.

题目大意：

给定一个数组，存储对应下标的权重。需要实现的是生成这个数组下标范围内的随机数，每个下标出现的次数与相应的权重相关。一道比较有意思而且有用的题。

数据范围：
  1. 1 <= w.length <= 10000
  2. 1 <= w[i] <= 10^5
  3. pickIndex will be called at most 10000 times.

样例1:

Input: ["Solution","pickIndex"] [[[1]],[]] Output: [null,0]

样例2:

Input: ["Solution","pickIndex","pickIndex","pickIndex","pickIndex","pickIndex"] [[[1,3]],[],[],[],[],[]] Output: [null,0,1,1,1,0]

题解：
# LeetCode
priority_queue与比较函数
DP求解子集的和一定时和的个数
